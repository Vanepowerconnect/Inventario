# Fleet Control 2026

Sistema de controle de frota e equipes para gerenciamento de assets, movimentações e inventário.

## Como usar

1. Abra o arquivo `index.html` no Chrome
2. Os dados ficam salvos automaticamente no navegador
3. Use o botão **💾 Salvar HTML** para gerar backup completo
4. Use o botão **⬇ Backup JSON** para exportar os dados

## Funcionalidades

- Dashboard com alertas e movimentações recentes
- Controle de movimentação de equipamentos
- Inventário completo com alertas de Verizon, E-pass, inspeção
- Histórico de movimentações
- Gestão de equipes por supervisor
- Cadastro completo de assets
- Gestão de pessoas (supervisores, teams, drivers)
- Relatórios imprimíveis por supervisor/team

## Estrutura

- `index.html` — aplicação completa (HTML + CSS + JS + dados)
